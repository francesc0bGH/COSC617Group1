## userauth branch
You'll need to copy the .env-boilerplate file to .env and add some secret to SESSION_SECRET.

For instance:
```zsh
cp .env-boilerplate .env
```

You'll also need to install the dependencies:

```zsh
npm install
```

Finally start the server via:

```
npm run devStart
```
